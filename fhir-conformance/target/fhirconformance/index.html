    Conformance Test Tool

    Information about Conformance Test Tool

    The FHIR Conformance Test Tool provides an easy way for servers to test their conformance to an implementation guide using provided test data and executing tests.

    Instructions to use Conformance Test Tool

    1. Download the Test Data in Excel or JSON Formats.
    2. Populate your system using the Test Data.
    3. Select "Authorize Server" or "No Authorization Required" and enter required details.
    4. Select the Implementation Guide and profiles/resources to run tests.
    5. Execute Tests.
    6. Examine Results.
