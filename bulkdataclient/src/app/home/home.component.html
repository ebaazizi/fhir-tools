Welcome to Bulk API Client

Information about the Bulk API client

Bulk API client can be used to view the list of Patients and select to create a panel that can be used for extraction. Once a panel is created use the Extract tab to retrieve the data for specific panels.

Instructions to use the Bulk API client

  1. Select Patient panel tab to view the list of patients.
  2. From the Patient panel select patients and create Group.
  3. Select the Extract tab to view the list of created Groups.
  4. Click on Extract button to get the bulk data of a particular Group.
  5. List of resources will be available to download on completion of Bulk data request.
